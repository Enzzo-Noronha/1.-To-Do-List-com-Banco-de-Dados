# 📝 To-Do List com Node.js e SQLite

Aplicação simples de lista de tarefas com persistência em banco de dados.

## 🚀 Tecnologias
- HTML, CSS, JavaScript
- Node.js
- Express
- SQLite3

## ⚙️ Funcionalidades
- Criar tarefas
- Listar tarefas
- Marcar como concluída
- Excluir tarefas

## 📦 Instalação

```bash
git clone https://github.com/Enzzo-Noronha/1.-To-Do-List-com-Banco-de-Dados.gitepo
cd backend
npm install
node server.js
